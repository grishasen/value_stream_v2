Six elements, one rule
Every tile shows enough to trust the number without leaving the tile.
  1. 1. Title + subtitle. The subtitle names the metric and the grain in one line.
  2. 2. Freshness pill. Green within SLA, amber stale, red on failure. Shows time since last successful run.
  3. 3. Chart. Plotly figure rendered from a metric query; downsampled with LTTB above 50k points.
  4. 4. Aggregate path. The exact physical aggregate the planner picked. Click to open in the SQL surface.
  5. 5. "Why these numbers?" Expander: processor, grain, scan stats, latest chunk, run id, config_hash, formula AST as text.
  6. 6. Sketch caveats. When a metric uses HLL or t-digest, the pill switches to "approx ±x" with a tooltip citing the bound from ALGORITHMS.md.
